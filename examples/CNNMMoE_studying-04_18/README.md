# 构建CNN+MMoE模型进行NIPTS回归预测

构建CNN+MMoE模型进行NIPTS346的回归预测，结果用于VCCA会议的摘要投稿。

## 结果记录

### 2024.04.19

模型构建完成，在同一测试集上获得的预测结果对比如下：

* ISO1999, RMSE=15.37
* CNN, RMSE=13.55
* CNN+MMoE, RMSE=13.52

以上成绩为训练结束前最后一轮训练结果在测试集上的预测成绩，尚未进行多个epoch训练以及保存最佳性能模型进行对比。

### 2024.04.22

优化模型训练方式，保存val_loss最新的模型作为最终结果。同一测试集上获得的预测结果对比如下：

* ISO1999, RMSE=15.37
* CNN, RMSE=13.52
* CNN+MMoE, RMSE=13.52
